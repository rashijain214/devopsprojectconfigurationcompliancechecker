## 🔹 Project Overview

### Problem Statement
Organizations struggle to ensure infrastructure follows security and configuration policies consistently. Manual verification is slow and error-prone.

### Objectives
- Detect configuration drift across infrastructure
- Enforce compliance policies using Puppet
- Monitor compliance violations using Nagios
- Generate reports and alerts for non-compliance

### Key Features
- Automated configuration validation
- Policy-based compliance checking
- Real-time monitoring dashboard
- Email alerting system
- Compliance reports generation
